# AppMod Blueprints Builder Skill

When working on CDK applications, chatbots, AI agents, document processing, RAG, knowledge bases, Bedrock agents, interactive agents, batch agents, or serverless AI infrastructure on AWS, ALWAYS activate the `appmod-blueprints-builder` skill before generating any code or design.

This applies to any task involving:
- AWS CDK constructs or stacks
- Conversational AI or chatbot infrastructure
- AI agent architectures on AWS
- Document processing pipelines
- RAG or knowledge base setups
- Bedrock agent configurations
- @cdklabs/cdk-appmod-catalog-blueprints usage
