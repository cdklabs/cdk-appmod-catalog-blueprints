  <action>
Create handler.py for the isolated script execution Lambda.

The handler must:

1. **Define constants:**
   ```python
   MAX_ROWS = 100000  # Hard limit per EXEC-04
   PREVIEW_ROWS = 100  # Default preview size
   ```

2. **Create handler(event, context) function:**
   ```python
   def handler(event, context):
       """
       Execute validated DataGenerator script and return schema + preview.

       Args:
           event: {script: str, row_count: int}
           context: Lambda context

       Returns:
           {success: bool, schema: dict, preview: list, row_count: int}
           or {success: False, error: str}
       """
   ```

3. **Extract and validate input:**
   ```python
   script = event.get('script', '')
   requested_rows = min(event.get('row_count', PREVIEW_ROWS), MAX_ROWS)

   if not script:
       return {'success': False, 'error': 'No script provided'}

   if requested_rows > MAX_ROWS:
       return {
           'success': False,
           'error': f'Row count exceeds maximum limit of {MAX_ROWS}'
       }
   ```

4. **Execute script in isolated namespace:**
   ```python
   # Create isolated execution namespace
   namespace = {}

   # Execute the script to define DataGenerator class
   exec(script, namespace)

   # Get the DataGenerator class
   if 'DataGenerator' not in namespace:
       return {'success': False, 'error': 'Script must define DataGenerator class'}

   DataGenerator = namespace['DataGenerator']
   ```

5. **Run DataGenerator and extract results:**
   ```python
   # Instantiate with preview row count
   generator = DataGenerator(num_rows=requested_rows)

   # Generate data
   datasets = generator.generate_datasets()
   schema = generator.generate_schema()

   # Convert first dataset to preview format
   if datasets and len(datasets) > 0:
       df = datasets[0]
       preview = df.head(PREVIEW_ROWS).to_dict(orient='records')
       actual_rows = len(df)
   else:
       preview = []
       actual_rows = 0
   ```

6. **Return structured response:**
   ```python
   return {
       'success': True,
       'schema': schema,
       'preview': preview,
       'row_count': actual_rows,
       'preview_count': len(preview)
   }
   ```

7. **Error handling:**
   - Wrap exec() in try/except for runtime errors
   - Catch AttributeError for missing methods
   - Return clear error messages with error type
   - Do NOT expose full stack traces (security)

8. **Import only allowed libraries at top:**
   ```python
   import json
   # pandas, numpy, faker are available in Lambda layer
   # but imported dynamically by the script
   ```

Note: This Lambda intentionally has NO boto3 imports or AWS SDK usage. It only executes the provided script using pandas/numpy/faker from the layer.

   ```typescript
   // =========================================================================
   // Execution Lambda - Isolated Script Runner
   // =========================================================================

   // Execution Lambda - minimal permissions, runs validated scripts
   const executionLambda = new PythonFunction(this, 'ExecutionLambda', {
     functionName: 'DataSynthExecutor',
     entry: path.join(__dirname, './resources/execution'),
     index: 'handler.py',
     handler: 'handler',
     runtime: Runtime.PYTHON_3_11,
     memorySize: 3072,  // 3GB for pandas operations
     timeout: Duration.minutes(5),
     description: 'Isolated Lambda for executing validated DataGenerator scripts',
   });

   // Note: NO additional IAM permissions granted
   // This Lambda only needs basic execution role (logs)
   // It has NO access to S3, DynamoDB, Bedrock, or other AWS services
   ```

3. **Add execute_script tool to InteractiveAgent:**
   ```typescript
   // Load execute_script tool (in addition to generate_script)
   const executeScriptTool = new Asset(this, 'ExecuteScriptTool', {

   });

   // Include both tools in InteractiveAgent
   tools: [generateScriptTool, executeScriptTool],

   // Update additionalPolicyStatementsForTools to include execution Lambda
   additionalPolicyStatementsForTools: [
     new PolicyStatement({
       effect: Effect.ALLOW,
       actions: ['lambda:InvokeFunction'],
       resources: [
         this.scriptGenerator.agentFunction.functionArn,
         executionLambda.functionArn,
       ],
     }),
   ],
   ```
