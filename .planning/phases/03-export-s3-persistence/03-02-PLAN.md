Tool signature per CONTEXT.md decisions:
```python
@tool
def export_dataset(script: str, row_count: int = 10000, user_id: str = "", session_id: str = "") -> Dict[str, Any]:
```

Parameters:
- `script`: Python DataGenerator script to execute
- `row_count`: Number of rows to generate (default 10,000)
- `user_id`: Cognito user sub for folder isolation (passed from agent context via Strands)
- `session_id`: Optional session ID for folder naming (generate UUID if not provided)

Implementation steps:
1. Initialize clients at module level (outside handler):
```python
lambda_client = boto3.client('lambda')
s3_client = boto3.client('s3')
```

2. Get environment variables:
```python
export_lambda_name = os.environ.get('EXPORT_LAMBDA_NAME')
bucket_name = os.environ.get('EXPORT_BUCKET_NAME')
```

3. Generate session_id if not provided:
```python
import uuid
if not session_id:
    session_id = str(uuid.uuid4())[:8]
```

4. Validate user_id (required for multi-tenant isolation):
```python
if not user_id:
    return {
        'success': False,
        'error': 'user_id is required for export. The agent should pass the authenticated user ID.',
        'recoverable': False
    }
```

5. Invoke export Lambda:
```python
payload = {
    'script': script,
    'row_count': row_count,
    'user_id': user_id,
    'session_id': session_id,
    'bucket_name': bucket_name
}
response = lambda_client.invoke(
    FunctionName=export_lambda_name,
    InvocationType='RequestResponse',
    Payload=json.dumps(payload)
)
```

6. Generate presigned URLs for each file (24-hour expiry per CONTEXT.md):
```python
PRESIGNED_URL_EXPIRY = 86400  # 24 hours in seconds

def generate_presigned_url(bucket: str, key: str) -> str:
    return s3_client.generate_presigned_url(
        'get_object',
        Params={'Bucket': bucket, 'Key': key},
        ExpiresIn=PRESIGNED_URL_EXPIRY
    )
```

7. Return structured response with presigned URLs:
```python
{
    'success': True,
    'downloads': {
        'csv': presigned_url_csv,
        'json': presigned_url_json,
        'schema': presigned_url_schema,
        'script': presigned_url_script
    },
    'row_count': result['row_count'],
    'summary': f"Your dataset is ready! Generated {result['row_count']} rows. Download links expire in 24 hours."
}
```

Error handling:
- Check for missing environment variables (graceful error message)
- Handle Lambda invocation errors
- Return recoverable=True for retry scenarios
